2. 實作exam    

exam內容: 設計出此 題目

步驟1: 基本零組件設計繪圖。

步驟2: 施力機構設計, 納入軌道下方控制盒中。

步驟3: 將軌道、鋼球與簡化後的控制盒輸入 CoppeliaSim。

步驟4: 配合控制系統設計, 納入感測器與施力機構或系統。

步驟5: 在 localhost 以 IPv6 ZMQ remote API Python 程式, 配合感測器作動對鋼球施力。

步驟6: 跨網路對協同組員的場景進行控制模擬。
